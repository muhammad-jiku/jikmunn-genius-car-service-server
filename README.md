
# jikmunn-genius-car-service-server

## Description
Backend server for Genius Car Service, a MERN stack project providing car service management features including authentication, service listing, and order processing.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT (JSON Web Token)
- dotenv
- CORS

## About
This project is the backend API for Genius Car Service. It handles user authentication, service management, and order processing. Designed for deployment on Vercel and built with scalability and security in mind.
